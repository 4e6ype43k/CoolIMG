# CoolIMG
### a very cool image format

>## **Notice**
> This simple image manipulation library was made purely as a **joke** and because I didn't want to make a .png reader. This library is **NOT** for day-to-day use (*unless you are brave enough*).  
> Be warned!

### What are the advantages of using this library?
- **Produces images that can be easily read by a human**
- **Lightweight**
- **No dependencies** (expect, of course, the standard C library)
- **Portable**
- **Very cool!**

### What are the disadvantages?
- **The actual image files take up way more space than an average png**
- **File read functions will crash if the path has no file**
- **Pixel data is stored on the heap** (though it isn't hard to alloc and free the memory)

## ***Use this at your own risk***
